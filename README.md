# ParkCloud CV Module

Прототип модуля компьютерного зрения для облачной платформы **ParkCloud**. 
Модуль решает задачу анализа видеопотока с камер на плоской парковке и выдачи в реальном времени событий о занятости парковочных мест.

## Архитектура решения

Модуль спроектирован как "черный ящик" и состоит из следующих компонентов (находятся в директории `models/`):

1. **`detector.py` (Детектор)**: Отвечает за поиск транспортных средств на кадре. Используется модель **YOLOv8** (`yolov8n.pt`). Детектор находит классы: *car, truck, bus, motorcycle* и возвращает уверенность (confidence) и координаты (bounding boxes).
2. **`calibrator.py` (Калибратор геометрии)**: Отвечает за преобразование перспективы (Bird's-eye view). Принимает координаты 4-х точек с камеры и рассчитывает матрицу гомографии (с помощью OpenCV). Центр нижней грани bounding box автомобиля проецируется на плоскую карту парковки.
3. **`analyzer.py` (Анализатор занятости)**: Сопоставляет спроецированные на "землю" координаты автомобилей с геометрией машиномест (заданы полигонами). Анализ пересечений выполняется с использованием библиотеки пространственной геометрии `shapely`.

## Структура проекта

```text
STAMP_test_task/
├── models/
│   ├── __init__.py
│   ├── analyzer.py       # Логика пересечения геометрии
│   ├── calibrator.py     # OpenCV преобразование перспективы
│   └── detector.py       # YOLOv8 распознавание
├── config.json           # Векторная карта: калибровочные узлы и парковочные места
├── main.py               # Главный пайплайн обработки реального кадра
├── test_pipeline.py      # Мок-скрипт для быстрого тестирования геометрии (без весов YOLO)

└── README.md             # Эта документация
```

## Установка и запуск

### Требования
* Git
* Python 3.8+

### Пошаговая установка

1. Склонируйте репозиторий на свой компьютер:
```bash
git clone https://github.com/m14el/stamp_cv.git
cd stamp_cv
```

2. Создайте виртуальное окружение:
```bash
# Для Windows:
python -m venv .venv
# Для macOS/Linux:
python3 -m venv .venv
```

3. Активируйте виртуальное окружение:
```bash
# Для Windows:
.venv\Scripts\activate
# Для macOS/Linux:
source .venv/bin/activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Работа с Датасетами и калибровкой

Для анализа реальных парковок есть папка `dataset/` . Все новые фотографии с камер видеонаблюдения нужно помещать туда.

### Инструкция по запуску:

Поскольку разные ракурсы камер имеют разное искажение пространства, программу нужно **откалибровать 1 раз для каждой уникальной камеры**, создав файл конфигурации.

#### Шаг 1. Визуальный инструмент калибровки и разметки
Для настройки геометрии и разметки парковочных мест нужно запустить инструмент на любой фотографии с нужной камеры:
```bash
python setup_calibration.py --image dataset/your_camera_first_image.jpg --config_out config_cam1.json
```
**Процесс внутри графического окна:**
1. **Калибровка (Перспектива):** Найдите на асфальте 4 точки, образующие ровный прямоугольник или квадрат в реальной жизни (например, линии парковочных мест). Кликните по ним (Верхний левый, Верхний правый, Нижний правый, Нижний левый). Нажмите `n` (next).
2. **Разметка мест:** Теперь вы видите плоскую проекцию "вид сверху". Кликайте по углам конкретного парковочного места, чтобы обвести его полигоном. Замкнули контур места? Нажмите `n` для сохранения и переходите к обводке следующего. 
3. **Сохранение:** Когда все места будут обведены, нажмите `s` для сохранения конфигурации в файл `config_cam1.json`. Окно закроется автоматически. 

#### Шаг 2. Запуск полного пайплайна (Анализ загруженности)
Используя созданный файл конфигурации, запустите основной анализатор на любом другом снимке с этой же камеры:

```bash
python main.py --config config_cam1.json --image dataset/your_image_to_test.jpg
```
*при самом первом запуске автоматически скачаются веса `yolov8n.pt`.*

**Ожидаемый вывод:** Пул событий для облака ParkCloud в формате JSON, содержащий % занятости для каждого отмеченного места.
```json
[
  {
    "camera_id": "dataset/your_image.jpg",
    "parking_id": "A1",
    "occupancy_percent": 100.0,
    "timestamp": "now"
  }
]
```

### Запуск тестов
Скрипт проверяет правильность проекции перспективы и логику пересечения зон парковки (создает пустое изображение `dataset/test_image.jpg` и эмулирует детекцию автомобиля).

```bash
python test_pipeline.py
```
